## Comandos de repositório

Iniciar diretório

```bash
git init
```

Verificar o status do reposítório

```bash
git status
```

Adicionar arquivos modificados

```
git add .
```

```bash
git add readme.md
```



Comitar

```bash
git commit -m "descrição"
```
